# Typing Speed Test

Ever wondered how fast and accurate you type? This web app lets you test your typing speed and accuracy in real time!  


## Live Demo
https://cyberyash24.github.io/Speed-Type-Testing/

## Features
- Displays a random sentence to type.
- Tracks **Words Per Minute (WPM)** as you type.
- Calculates **typing accuracy** instantly.
- Saves your **last score** using browser storage so you can see your previous performance.
- Simple and clean interface — just start typing and watch your stats update!

## How to Use
1. Click the **Start** button to begin the test.
2. Type the sentence displayed in the box.
3. WPM and accuracy will update as you type.
4. Click **Restart** to try again.
